# Estrutura de Dados

## Trabalho 1

Na pasta [Trabalho 1]("https://github.com/mauro-moura/estrutura_de_dados/tree/main/Trabalho%201"), há todo o conteúdo para a execução da primeira tarefa, sendo ela:

Dado um arquivo (contido na pasta `./Trabalho 1/matrices`) contendo uma ilha, onde a primeira linha contem o número de linhas, a segunda é o número de colunas, e as outras são conteúdos de uma matriz que representa uma ilha. Nesta ilha há um gerador vazando óleo, representado por 2, está poluindo a ilha, e vaza para a terra, representada por 1, mas é considerado que a água, representada por 0, absorve totalmente o óleo.

Um modelo de arquivo de ilha é mostrado abaixo:

```txt
5
6
000000
011100
012100
011100
000000
```

O arquivo `run_main.sh` é capaz de compilar o código, gerar um arquivo de saída, executá-lo e apagá-lo. Assim, é possível executar no Linux utilizando o comando:

```bash
./run_main.sh
```
